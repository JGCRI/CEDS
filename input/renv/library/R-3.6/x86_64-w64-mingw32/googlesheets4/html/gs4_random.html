gs4_random {googlesheets4}R Documentation

Generate a random Sheet name

Description

Generates a random name, suitable for a newly created Sheet, using ids::adjective_animal().

Usage

gs4_random(n = 1)

Arguments

n

Number of names to generate.

Value

A character vector.

Examples

gs4_random()
