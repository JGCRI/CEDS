isFALSE {xfun}R Documentation

Test if an object is identical to FALSE

Description

A simple abbreviation of identical(x, FALSE).

Usage

isFALSE(x)

Arguments

x

An R object.

Examples

library(xfun)
isFALSE(TRUE)  # false
isFALSE(FALSE)  # true
isFALSE(c(FALSE, FALSE))  # false
