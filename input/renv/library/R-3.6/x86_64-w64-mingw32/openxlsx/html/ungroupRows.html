ungroupRows {openxlsx}R Documentation

Ungroup Rows

Description

Ungroup a selection of rows

Usage

ungroupRows(wb, sheet, rows)

Arguments

wb

A workbook object

sheet

A name or index of a worksheet

rows

Indices of rows to ungroup

Details

If row was previously hidden, it will now be shown

Joshua Sturm
