char {pillar}R Documentation

Format a character vector in a tibble

Description

These functions are reexported as tibble::char() and tibble::set_char_opts().

Usage

char(
  x,
  ...,
  min_chars = NULL,
  shorten = c("back", "front", "mid", "abbreviate")
)

set_char_opts(
  x,
  ...,
  min_chars = NULL,
  shorten = c("back", "front", "mid", "abbreviate")
)
